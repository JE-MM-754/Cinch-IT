# AI Kanban Dashboard - MVP Phase 1

A simple, clean kanban board for managing CinchIT, MetaForge, and Job Search projects.

## 🚀 Live Demo

**Production URL:** https://ai-kanban-dashboard.vercel.app

## ✅ Features Implemented

### Core Functionality
- ✅ **4-Column Kanban Board:** Ideas, Start/Launch, In Progress, Complete
- ✅ **Drag & Drop:** Full drag and drop functionality between columns using @dnd-kit
- ✅ **Mock Project Data:** Sample tasks for CinchIT, MetaForge, and Job Search projects

### Design & UX
- ✅ **Clean, Professional UI:** Modern design with Tailwind CSS
- ✅ **Project Color Coding:** Blue (CinchIT), Green (MetaForge), Purple (Job Search)  
- ✅ **Priority Indicators:** High/Medium/Low with color-coded badges
- ✅ **Task Details:** Title, description, project tags, and category tags
- ✅ **Responsive Layout:** Works on desktop and mobile
- ✅ **Task Count:** Shows number of tasks per column

### Technical Stack
- **Framework:** Next.js 16+ with TypeScript
- **Styling:** Tailwind CSS
- **Drag & Drop:** @dnd-kit/core, @dnd-kit/sortable, @dnd-kit/utilities
- **Deployment:** Vercel (Production)
