## Risk Assessment

### 🚨 CRITICAL (1)
1. **Git Exposure Risk**
   - **Threat:** Accidental `git add .` would stage Discord token, API keys, financial records
   - **Impact:** Complete identity/financial compromise if pushed to public repo
   - **Likelihood:** HIGH (common developer mistake)
   - **Remediation:** Update .gitignore immediately, add pre-commit hooks

### 🔥 HIGH (1) 
1. **Sensitive Data in Plaintext**
   - **Threat:** TOOLS.md contains API keys, not protected by .gitignore
   - **Impact:** API key theft, unauthorized access to Motion, browser relay
   - **Likelihood:** HIGH if git repo is ever shared
   - **Remediation:** Move to environment variables, add TOOLS.md to .gitignore

### ⚠️ MEDIUM (1)
1. **Session Transcript Privacy**
   - **Threat:** Detailed financial/personal transcripts accessible
   - **Impact:** Privacy breach if system is compromised
   - **Likelihood:** LOW (local access required)
   - **Remediation:** Add transcript patterns to .gitignore
